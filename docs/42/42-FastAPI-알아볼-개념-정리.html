FastAPI 알아볼 개념 정리

탐구 생활/FastAPI

2024-11-19 13:16:14

원래 SpringBoot & Java 로 백엔드 어플리케이션을 개발하고 있었다. 그런데 최근 회사 테크 리드께서 FastAPI&Python 그리고 Cloud Native로 대 변혁을 꽤하고 있는바...나는 조용히 새로운 기술 스택을 공부해서 잘 팔로업 하는게 소임을 다하는 것이라고 생각이 든다.

 

그래서 FastAPI 를 어떻게 공부할까 고민하던중 기존에 이용하던 SpringBoot 의 개념과 대응되는(1:1 로 대응이 되진 않더라도...) 개념을 FastAPI 에서 찾아보면 좋겠다는 생각이 들었다.

 

그러한 이유로 이 글에서는 SpringBoot 에서 주로 이용했던 기능을 FastAPI 에서는 어떻게 구현했는지 알아보는 세부 페이지에 대한 목차 정도가 될 것이다. 이 글은 즉  어떤 개념을 알아봐야하는지 구조화회는 기능을 수행한다고 보면 되겠다.

 

SpringBoot 개념 FastAPI 개념
application.yml (환경변수 주입)  
DI (의존성 주입과 관리)  
가장 잘 알려진 프로젝트 구조 (Controller, Service, Repository)  
Filter & Interceptor  
HttpRequest 접근  
path variable, query param, request body, multipart data  
request validation  
advice controller (중앙집중형 예외 관리)  
JdbcTemplate (DB conenction, law query)  
ORM  
Transation Control (propagation, rollback 등)  
Application Event (다른 쓰레드에서 비동기적 작업 수행)  
AOP (비즈니스 로직과 관심사 분리)  
Actuator (상태점검, 모니터링)  
Thymeleaf (UI 템플릿)  
Test (Integartion, Slice, Unit 테스트)
